# Eagle Taxi

## 프로젝트 개요

- 서비스명: Eagle Taxi
- 한글명: 독수리 택시
- 팀명: Mirae Mobility
- 제품 버전: `v0.1.0-alpha`
- 개발 형태: 캠퍼스 이동을 위한 택시 파티 중개 MVP
- 대상 사용자: 연세대학교 미래캠퍼스 학생
- 핵심 가치: 이동 조건 입력. 유사 파티 추천. 파티 생성. 파티 참여. 채팅 조율. 관리자 운영.

## 서비스 설명

Eagle Taxi는 출발지와 목적지와 희망 출발 시간을 기준으로 같은 방향의 학생을 연결하는 택시 파티 중개 서비스입니다.

사용자는 빠른 시작에서 이동 조건을 입력합니다. 조건과 비슷한 파티가 있으면 추천 결과를 확인합니다. 원하는 파티가 없으면 같은 조건으로 파티 목록을 보거나 새 파티를 만들 수 있습니다.

서비스는 실제 택시 호출과 실제 결제와 운전자 연결을 제공하지 않습니다. 학생 간 이동 파티를 만들고 관리하는 MVP 범위를 우선 구현합니다.

## 로컬 실행

```bash
./run-dev-mac.sh
```

Windows 환경에서는 다음 파일을 실행합니다.

```bat
run-dev.bat
```

macOS 환경에서는 통합 런처 앱을 실행할 수도 있습니다.

```text
EagleTaxiLauncher.app
```

## 개별 실행

백엔드 실행:

```bash
cd backend
python3 -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

사용자 프론트 실행:

```bash
cd frontend
npm install
npm run dev -- --host 127.0.0.1 --port 5174
```

관리자 콘솔 실행:

```bash
cd admin
npm install
npm run dev -- --host 127.0.0.1 --port 5173
```

## 현재 개발 버전 메모

- 랜딩 페이지는 서비스 소개형 구조로 재구성되었습니다.
- 서비스 배경과 서비스 소개와 이용 방법과 개발자 정보가 포함됩니다.
- 이용 방법 섹션은 단계별 인터랙션 애니메이션을 포함합니다.
- 로그인 후 서비스 메인은 빠른 시작 중심으로 구성됩니다.
- 파티 목록은 조건 입력 전에도 전체 목록을 둘러볼 수 있습니다.
- 문의와 신고는 사용자 화면과 관리자 콘솔에서 대화형으로 관리됩니다.
- 관리자 로그인과 문의 상태 변경은 조작 기록에 남습니다.
- 비밀번호 해시는 직접 `bcrypt` 검증 방식으로 처리합니다.
- 통합 런처는 서버 실행과 외부 연결과 로그 확인을 하나의 화면에서 관리합니다.
- 백엔드는 접속자 확인을 위해 `[ACCESS]` 형식의 접근 로그를 출력합니다.
- 서비스 프론트와 관리자 콘솔은 외부 도메인 접속을 허용하도록 Vite 설정이 보완되었습니다.
- 로컬 DB와 로컬 환경 파일은 배포 대상에서 제외합니다.
